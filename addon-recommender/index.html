Firefox Add-ons recommender

This is a simple proof concept of a recommendation engine for Firefox, see Bug 1284280.

The model is trained with the IDs of the add-ons reported by the users, both legacy and using webextensions. Add-ons which are not publicly reported by the addons.mozilla.org web API are excluded from the model. The page will only suggest non-legacy addons.

Know issues and future work
